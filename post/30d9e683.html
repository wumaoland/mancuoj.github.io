Notes

指令 说明
git init 初始化本地文件夹为一个git仓库
git clone [url] copy ~~ 🔥
git remote add origin [url].git 关联本地仓库与远端仓库
git add . 将修改放入暂存区
git status 查看所有变更的文件
git commit -am "update~" 提交并添加comment，‘-a’可跳过add步骤
git push -u origin master 推送到远端库，‘-u’后续不用再写origin master
😇😇😇
git pull 从远端获取最新版本并merge
git fetch 从远端获取最新版本不自动merge
git branch 查看本地所有分支
git reset HEAD [file] 取消之前的add
git rm [file] 删除已跟踪文件
git log 查看提交历史
